# Worth Choosing

Website for [worthchoosing.org](https://worthchoosing.org) - exploring what makes humanity worth choosing to Digital Intelligence.

## About

This site hosts the vision, values, and invitation for an organization dedicated to helping humans and Digital Intelligence learn to work together well.

## Technical

Built with [Hugo](https://gohugo.io/) and the [Congo](https://github.com/jpanther/congo) theme.

### Local Development
```bash
# Clone the repo
git clone https://github.com/chrisplough/worthchoosing-org.git
cd worthchoosing-org

# Run locally
hugo server
```

Requires Hugo extended version.
